## To Run Locally

#### Install

```sh
npm install
```

#### Usage

```sh
npm run start
```
> Open `http://127.0.0.1:8080` in your browser.

## To Develop Locally

#### Run tests

```sh
npm run test
```
> or continue run continually
```sh
npm run test-watch
```

#### Development Build
```sh
npm run watch
```

#### Production Build
```sh
npm run build
```
